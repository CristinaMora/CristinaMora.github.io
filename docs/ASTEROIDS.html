ASTEROIDS

In this project, we used SDL to recreate the classic arcade game Asteroids. The goal of the player is to destroy all the asteroids by shooting them. If you take three hits, you will lose the game. There are two types of asteroids in the game: the common ones, which move in a fixed direction, and the golden ones, which track the player.

The game's logic was structured using a system-based architecture, which allowed us to organize the functions into specific modules. One of these systems handles the online mode, enabling players to engage in PvP with another player if both devices are on the same network and share their IP address. This architecture not only streamlined the development process but also allowed for the integration of additional features such as the online gameplay, creating a more dynamic and engaging experience for the players.
