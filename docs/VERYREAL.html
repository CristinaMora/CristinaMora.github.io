VERY REAL

Very Real is a modular game engine written in C++ and built from scratch with Ogre for rendering, SDL for window and input management, FMOD for audio, Bullet for physics simulation, and Lua for level scripting.

The engine is designed to run games dynamically by loading them as DLLs at runtime. This ensures a full decoupling between engine and game logic, so developers can build games without modifying the engine source. The DLLs are automatically linked after the engine initializes the necessary systems and singletons.

Key Modules

  • Rendering: OGRE-based system with full 3D support, cameras, lights, meshes, shaders and UI overlays.
  • Audio: FMOD-powered 2D/3D sound system with positional audio and listener tracking.
  • Input: SDL-based input module supporting mouse, keyboard, and controllers.
  • UI: Overlay-driven interface components including buttons, text, and progress bars.
  • Physics: Bullet physics integration for rigid bodies and collisions.
  • Lua Scripting: Lua integration used for dynamic level loading and component-based configuration.
  • ECS Architecture: Core built on a dynamic Entity-Component-System model with modular creation via factory classes.
