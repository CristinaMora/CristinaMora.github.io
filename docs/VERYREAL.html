VERY REAL

Very Real is a video game engine developed from scratch using C++ and well-known libraries such as Ogre, SDL, FMOD, and Bullet. Its modular and independent architecture separates the engine entirely from the games it runs. The games are implemented as DLLs (Dynamic Link Libraries), which the engine dynamically loads at runtime, allowing developers to focus solely on creating their games without worrying about engine integration.

The engine comprises several independent modules, such as input handling, rendering, physics, and sound, ensuring its versatility and reusability in future projects. To validate its functionality, we developed two 3D games:

  • Ogreman: A horror game inspired by Slenderman. Players must collect notes to uncover the story while escaping from Ogreman, a patrolling enemy that switches to pursuit mode using an A* algorithm when it detects noise or sees the player.
  • Oveja Vegana: A survival game inspired by Brotato. Players control a sheep that must fend off waves of hunters by shooting at them and using medkits to heal when necessary.

This project was a collaborative effort by a team of 11 people, where we applied pair programming and held weekly meetings to coordinate tasks and track progress. My main responsibility was the rendering module, and I later implemented an automatic compilation system for the engine and contributed to the game development.

Very Real represents a significant technical milestone and an invaluable experience in teamwork, software development, and game engine design.
