VERY REAL

Very Real is a custom-built game engine developed from scratch in C++ using libraries like Ogre, SDL, FMOD, and Bullet. Its modular and decoupled architecture allows for a complete separation between the engine and the games it runs.

Games are implemented as DLLs (dynamic libraries), which the engine loads at runtime, enabling developers to focus solely on building their game without worrying about engine integration.

The engine consists of modules such as rendering, physics, audio, and input, which are reusable for future projects. To validate its functionality, we developed two 3D games:

  • Ogreman: A horror game inspired by Slenderman. The player collects notes while avoiding Ogreman, who patrols and chases the player using an A* algorithm when noise is heard or the player is spotted.
  • Oveja Vegana: A survival game inspired by Brotato. You control a sheep that defends itself from hunters by shooting and healing using medkits.

The project was developed by a team of 11 members using pair programming and weekly meetings. My main responsibility was the rendering module, along with implementing an automatic build system for the engine and contributing to game development.

Very Real was a major technical milestone and a key experience in teamwork, software development, and engine design.
